HTML Styling

HTML Styles are used to style HTML elements. It also means changing default values or creating custom values

For instance, styling can change the default values of text color as black, background color as white, text alignment as left and text size as 12 pixels.

Internal Styling

Using an internal style sheet is also called internal styling.

An Internal Style Sheet is composed of one or more Cascading Style Sheet (CSS) rule set.

A CSS rule-set consists of a selector and a declaration block surrounded by curly braces that contain one or more CSS dclarations separated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

They are all enclosed inside the <style> element with it's type="text/css" attribute which is included inside the <head> element.

Internal Style Sheet Syntax

  <style type="text/css">

    element {

      property: value;

    }

  </style>

There will be a lot of internal styling examples throughout this entire tutorial so sit tight and enjoy the ride 😎

Example:

<!DOCTYPE html>
<html lang ="en">
 <head>
  <style type="text/css">
   h1 {
    color: green;
   }
   p {
    font-size: 14px;
   }
  </style>
 </head>

 <body>
  <h1>This is a heading</h1>
  <p>This is a paragraph</p>
 </body>
</html>

Output:

I am a heading

I am a paragraph


Inline Styling

Inline styling is used to style elements using the style attribute with CSS declarations inside which are similar to internal styling.

Inline Styling Syntax

<element style="property: value; property: value">

Background Color Example:

<html lang ="en">
 <head>
  <title>Inline Styling</title>
 </head>
 <body style="background-color: purple;">
  <!-- content goes here -->
 </body>
</html>

Output:



Text Color Example:

<p style="color: blue;">I am a blue text</p>
<p style="color: red;">I am a red text</p>
<p style="color: green;">I am a green text</p>
<p style="color: yellow;">I am a yellow text</p>

Output:

I am a blue text

I am a red text

I am a green text

I am a yellow text



Text Font-Family Example:

<p style="font-family: sans-serif;">I am a sans-serif text</p>
<p style="font-family: times new roman;">I am a times new roman text</p>
<p style="font-family: courier;">I am a courier text</p>
<p style="font-family: cursive;">I am a cursive text</p>

Output:

I am a sans-serif text

I am a times new roman text

I am a courier text

I am a cursive text



Text Size Example:

<p style="font-size: 15px;">My size is 15 pixels</p>
<p style="font-size: 20px;">My size is 20 pixels</p>
<p style="font-size: 25px;">My size is 25 pixels</p>
<p style="font-size: 30px;">My size is 30 pixels</p>

Output:

My size is 15 pixels

My size is 20 pixels

My size is 25 pixels

My size is 30 pixels



Text Align Example:

<p style="text-align: left;">I am aligned to the left</p>
<p style="text-align: center;">I am aligned to the center</p>
<p style="text-align: right; color: red;">I am aligned to the right and my color is red</p>

Output:

I am aligned to the left

I am aligned to the center

I am aligned to the right aand my color is red
