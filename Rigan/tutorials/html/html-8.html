HTML Colors

HTML Colors define colors that we see on a web page.

In this chapter, we will use inline styling with CSS properties color and baackground-color to change the text color and/or background color of HTML Elements.

HTML Colors can be defined using the following:

  • Color Name: It defines any valid color by its color name e.g white, blue.
  • Hex Color Value: Or Hexadecimal Color Value. It defines any valid color by a hash followed by letters and/or numbers e.g. #ffffff(white), #000000(black).
  • RGB Color Value: RGB stands for Red, Green & Blue. It defines any valid color by this format "rgb(number, number, number)" e.g. rgb(255, 255, 255), rgb(0, 0, 0). The numbers represent the intensity of the red, green aand blue colors in that particular order. The numbers range from 0 to 255.

Example:

<!-- changing background color using a color name -->
<div style="background-color: black">

  <!-- changing text color using a hexadecimal value -->
  <p style="color: #008080;">I am a text in Teal color</p>

  <!-- changing text color using an RGB value -->
  <p style="color: rgb(0, 128, 128);">I am a text in Teal color</p>

</div>

Output:

I am a text in Teal color

I am a text in Teal color
