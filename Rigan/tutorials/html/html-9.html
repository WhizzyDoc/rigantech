HTML Responsive

To make an HTML page responsive, simply add this inside the <head> element.

<meta name="viewport" content="width=device-width, initial-scale=1" />

Explanation:

The meta name="viewport" defines the name of the Meta data as Viewport (the available display screen on a device)

The content defines the propeties/description for the Meta data

The width=device-width sets the width of the webpage to conform to the width of the device

The initial-scale=1 sets the initial size of webpage to original size

Example:

<!DOCTYPE html>
<html lang ="en">
 <head>
  <title>Hello World!</title>
  <meta name="viewport" content="width=device-width, initial-scale=1" />
 </head>
 <body>
  ...
  ...
 </body>
</html>
