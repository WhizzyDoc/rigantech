Where can I edit/create HTML files?

In a text editor! Creating HTML files is free. You don't need to download expensive applications to do so.

Look at the list below for some free apps/softwares you could use to edit HTML files.

  • Visual Studio Code (PC)
  • Notepad++ (PC)
  • Sublime Text Editor (PC)
  • ATOM (PC)
  • TrebEdit (Android App)
  • Code Editor (Android App)
  • SPCK Editor (Android App)
  • DroidEdit (Android App)
  • This Website! We have an in-built code editor with syntax highlighting and can save files too.

Those are the most commonly used text editors for creating HTML files.

You can download any application you want.

Saving HTML Files

You need to save your HTML files with the .html file extension. For example, newfile.html.

If you don't save your files with the .html file extension, you won't be able to run it on browsers.

Example:

<p>Hello World!</p>

Output:

Hello World
